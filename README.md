## How to start fabric
```
./startFabric.sh
./createPeerAdminCard.sh
```

Show playground:
```
cd hyperledger-composer/tutorial-network
../composer-playground
```

Start Rest Server:
```
cd hyperledger-composer/tutorial-network
../composer-rest-server
```
