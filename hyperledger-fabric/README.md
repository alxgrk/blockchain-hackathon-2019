# Docker

## How to start fabric using docker 

```
docker-compose -d up
```

# How to clean fabric using docker
```
docker-compose down
```

# Manually

## How to start fabric manually

```

./createPeerAdminCard.sh
```

# How to clean fabric manually

```

```
